## Abstract

This project is for the team project of 26SPR-CIS-11-22428 at Moreno Valley College, lead by Kasey Nguyen.
It contains LC-3 assembly source code for the end-of-semester project, of which Option C: "Character Counter for Names" was selected.
Keanu Arao, Justin Sweers, and Suleny Perez are authors of this code.



## Input/Output

The input allows for 100 characters, of which, the letters A-Z are allowed of either capitalization, plus the space character.
The output is a [Frequency Distribution](https://www.geeksforgeeks.org/maths/frequency-distribution/) of the letters from the input, along with their according number of appearances, newline delimited, and in alphabetical order.



## Usage Instructions

Only the source code is provided, you must assemble from source yourself using LC-3's assembler. Additionally, you must use LC-3's official simulator to run the program. Both programs for doing this can be found [here](https://highered.mheducation.com/sites/0072467509/student_view0/lc-3_simulator.html).
